
# 💰 Smart Expense Tracker Dashboard

## Overview

Smart Expense Tracker Dashboard is a web-based personal finance management application designed to help users track daily expenses, monitor spending habits, and gain valuable financial insights through interactive visualizations. The system simplifies expense management by organizing transactions and generating meaningful reports for better budgeting and decision-making.

## Features

* Daily Expense Tracking
* Category-wise Expense Management
* Monthly Expense Reports
* Interactive Dashboard
* Real-Time Data Visualization
* Spending Trend Analysis
* Budget Monitoring
* Responsive User Interface
* Local Data Storage

## Technology Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Data Visualization

* Chart.js

### Storage

* Browser Local Storage

### Expense Management

## Installation

1. Clone the repository

```bash
git clone https://github.com/adityaarunsabat-art/Smart-Expense-Tracker-Dashboard.git
```

2. Open the project folder

3. Run the application by opening `index.html` in your browser
